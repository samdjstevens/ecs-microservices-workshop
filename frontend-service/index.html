Adding the service with CDK#

Once more, we need another CDK dependency for our stack. This time it's the @aws-cdk/aws-ecs-patterns package, which gives us some ready-made "recipies" for building things with ECS.

Stop the npm run watch command and run the following command:

npm install @aws-cdk/aws-ecs-patterns --save && npm run watch
import * as cdk from '@aws-cdk/core';
import * as ec2 from "@aws-cdk/aws-ec2";
import * as ecs from "@aws-cdk/aws-ecs";
import * as ecr from "@aws-cdk/aws-ecr";
import * as iam from "@aws-cdk/aws-iam";
import * as servicediscovery from "@aws-cdk/aws-servicediscovery";
import * as ecs_patterns from "@aws-cdk/aws-ecs-patterns";
import { CfnOutput } from '@aws-cdk/core';
